This guide walks you through making your first API call to the Thrad SSP. You'll use sandbox mode so no real auction runs and an ad is always returned.

## Prerequisites

- A Thrad account. Go to [platform.thrad.ai](https://platform.thrad.ai) and sign up as a **developer**.

<Note>
  The platform is currently open to trusted partners. After you apply, allow up to 24 hours for your account to be reviewed before you can create an API key.
</Note>

- Your API key from the Thrad platform.

## Make your first request

Send a POST request with a sample conversation. Set `production: false` to use sandbox mode.

<CodeGroup>

```bash cURL
curl -X POST https://ssp.thrad.ai/api/v1/ssp/bid-request \
  -H "thrad-api-key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "user_test-123",
    "chatId": "chat_test-456",
    "messages": [
      {
        "role": "user",
        "content": "I am looking for running shoes under $150",
        "timestamp": "2025-01-01T10:00:00Z"
      },
      {
        "role": "assistant",
        "content": "I can help you find great running shoes in that budget. Do you prefer road or trail running?",
        "timestamp": "2025-01-01T10:00:05Z"
      }
    ],
    "production": false
  }'
```

```javascript JavaScript
const response = await fetch('https://ssp.thrad.ai/api/v1/ssp/bid-request', {
  method: 'POST',
  headers: {
    'thrad-api-key': 'your-api-key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    userId: 'user_test-123',
    chatId: 'chat_test-456',
    messages: [
      {
        role: 'user',
        content: 'I am looking for running shoes under $150',
        timestamp: new Date().toISOString()
      },
      {
        role: 'assistant',
        content: 'I can help you find great running shoes in that budget. Do you prefer road or trail running?',
        timestamp: new Date().toISOString()
      }
    ],
    production: false
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    "https://ssp.thrad.ai/api/v1/ssp/bid-request",
    headers={
        "thrad-api-key": "your-api-key",
        "Content-Type": "application/json",
    },
    json={
        "userId": "user_test-123",
        "chatId": "chat_test-456",
        "messages": [
            {
                "role": "user",
                "content": "I am looking for running shoes under $150",
                "timestamp": "2025-01-01T10:00:00Z",
            },
            {
                "role": "assistant",
                "content": "I can help you find great running shoes in that budget.",
                "timestamp": "2025-01-01T10:00:05Z",
            },
        ],
        "production": False,
    },
    timeout=5,
)

data = response.json()
print(data)
```

</CodeGroup>

## Handle the response

In sandbox mode you'll always receive a bid. In production, `bid` may be `null`.

```json
{
  "requestId": "api_req_123",
  "timestamp": "2025-01-01T10:00:01Z",
  "totalTime": 0.123,
  "status": "success",
  "message": "Bid successful",
  "data": {
    "bid": {
      "price": 5.50,
      "advertiser": "Nike",
      "headline": "Premium Running Shoes",
      "description": "Perfect for marathon training. Lightweight and comfortable.",
      "cta_text": "Shop Now",
      "url": "https://ssp.thrad.ai/api/v1/dsp/redirect/abc123",
      "image_url": "https://cdn.example.com/nike-shoes.jpg",
      "dsp": "thrad_dsp",
      "bidId": "bid_abc123"
    }
  },
  "error": null
}
```

Check `data.bid` before rendering. A `null` bid is not an error — it means no DSP won the auction.

```javascript
if (data.data.bid) {
  renderAd(data.data.bid);
} else {
  // No ad this turn — continue normally
}
```

## Base URL and rate limits

| Property | Value |
|----------|-------|
| Base URL | `https://ssp.thrad.ai` |
| Rate limit | 100 requests per second per API key |
| Max response time | ~3.5 seconds |

Rate limit headers are included in every response:

```
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1234567890
```
