## 1. Get your API key

Create a developer account at [platform.thrads.ai](https://platform.thrads.ai). Once your account is approved (up to 24 hours), generate an API key from your dashboard.

<Note>
  The platform is currently open to trusted partners. After you apply, allow up to 24 hours for your account to be reviewed.
</Note>

## 2. Authenticate

Pass your API key in the request header:

```
thrad-api-key: your-api-key
```

If you're integrating from a browser, register your domain with Thrads for CORS.

## 3. Try it in sandbox mode

Set `production: false` — an ad is always returned so you can test without any live auction.

<CodeGroup>

```bash cURL
curl -X POST https://ssp.thrads.ai/api/v1/ssp/bid-request \
  -H "thrad-api-key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "user_123",
    "ad_format": "sponsored_prompt",
    "country_code": "US",
    "production": false
  }'
```

```javascript JavaScript
const response = await fetch('https://ssp.thrads.ai/api/v1/ssp/bid-request', {
  method: 'POST',
  headers: {
    'thrad-api-key': 'your-api-key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    userId: 'user_123',
    ad_format: 'sponsored_prompt',
    country_code: 'US',
    production: false
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    "https://ssp.thrads.ai/api/v1/ssp/bid-request",
    headers={
        "thrad-api-key": "your-api-key",
        "Content-Type": "application/json",
    },
    json={
        "userId": "user_123",
        "ad_format": "sponsored_prompt",
        "country_code": "US",
        "production": False,
    },
    timeout=5,
)

data = response.json()
print(data)
```

</CodeGroup>

You'll get back a response with a `bids` array:

```json
{
  "status": "success",
  "data": {
    "bids": [
      {
        "bidId": "sp_bid_abc123",
        "advertiser": "Nike",
        "logo_url": "https://cdn.example.com/nike-logo.png",
        "image_url": "https://cdn.example.com/nike-shoes.jpg",
        "sponsored_prompt": "What are the best running shoes for marathons?",
        "append_message": "Check out the Nike Air Zoom Alphafly — the marathon record-holder's shoe.",
        "cta_text": "Shop Now",
        "url": "https://ssp.thrads.ai/api/v1/tracking/redirect?token=...",
        "view_url": "https://localhost:8000/api/v1/tracking/view?token=...",
        "sponsored_prompt_click_url": "https://events.thrad.ai/api/v1/tracking/sponsored-prompt-click?token=...",
        "placement": "sponsored_prompt"
      }
    ]
  }

```

In production, `bids` may be empty when no ad matches the context — that's expected, not an error.
